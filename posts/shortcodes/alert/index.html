Alert Shortcode

Oct 22, 2020 #Shortcode#用戶手冊#Alert

本文展示瞭如何使用 alert shortcode。

1{{< alert "Message" [type] >}}

The parameter type is optional. Default to info.

Info

1{{< alert "Info" >}}

1$ curl https://test.com
2$ openshift-install
Info

Success

1{{< alert "Success" success >}}
Success

Warning

1{{< alert "Warning" warning >}}
Warning

Danger

1{{< alert "Danger" danger >}}
Danger
